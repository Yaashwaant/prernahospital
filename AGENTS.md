## Project Summary
This project is a high-fidelity clone of the Prerna Hospital website homepage, built with Next.js and Tailwind CSS. It features a clean, professional medical aesthetic with a teal and blue color palette, responsive navigation, and interactive hero and service sections.

## Tech Stack
- Framework: Next.js 15
- Styling: Tailwind CSS
- Icons: Lucide React
- Animations: Framer Motion
- Components: React (Functional)

## Architecture
- `src/app/page.tsx`: Main entry point assembling the components.
- `src/components/HospitalHeader.tsx`: Top bar and main navigation.
- `src/components/HeroSection.tsx`: Gradient hero area with image cards and stats.
- `src/components/ServiceCards.tsx`: Grid of medical specialty cards.

## User Preferences
- High fidelity to the provided screenshot.
- Professional medical aesthetic.
- Interactive elements using Framer Motion.

## Project Guidelines
- Use Teal (#008489) and Dark Blue (#003D52) as primary colors.
- Maintain a clean, white-space heavy layout.
- Ensure responsiveness across mobile and desktop.

## Common Patterns
- Rounded corners (2xl to 4xl) for cards and sections.
- Soft shadows for elevated elements.
- Gradient backgrounds for high-impact sections.
